ML Internship Projects - February
This repository contains machine learning projects and exploratory data analysis (EDA) completed during a 20-session internship program. The work covers Python fundamentals, data visualization, and predictive modeling

Project 1: Sales Analysis and Profit Prediction
  An analysis of the Superstore Sales Dataset (9,994 rows) to understand retail trends and predict financial outcomes.
  Objective: Perform EDA and train a model to predict profit based on sales-related variables
  Key Insights: * The Technology category generates the highest sales.
  Sales are relatively balanced across regions, with the South and East slightly higher.
  Model: Linear Regression using scikit-learn.Evaluation: Measured using Mean Squared Error (MSE) and R^{2} Score.


Project 2: Student Performance Analysis
   A dual-part project aimed at categorizing and predicting academic success based on study habits and attendance.
   
  Part A: Multinomial Classification
   Objective: Classify students into Low, Medium, or High performance categories .
   Model: Multinomial Logistic Regression.
   Results: Achieved an accuracy score of approximately 0.636.

  Part B: Regression
   Objective: Predict a student's numerical Final Grade.
   Model: Linear Regression.
   Evaluation: R^{2} Score of -0.022 and Mean Absolute Error (MAE) of 8.32.

   Skills & Tools Covered
   Data Science Libraries: * NumPy: Array manipulation and reshaping.
   Pandas: DataFrame operations, filtering, and handling missing values.
   Visualization: Matplotlib, Seaborn (Titanic/Netflix EDA), and interactive Plotly charts.
   Machine Learning: Linear and Logistic (Binary/Multinomial) Regression workflow.
